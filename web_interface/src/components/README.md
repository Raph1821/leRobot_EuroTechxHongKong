# Components

This directory will contain the UI components:

- `RobotModel3D.ts` — Three.js kinematic chain with STL meshes (Task 7.4)
- `JointControlPanel.ts` — Slider controls for arm joints (Task 8.1)
- `GripperControl.ts` — Gripper slider/buttons (Task 8.2)
- `PoseManager.ts` — Save/load/delete poses, trajectory playback (Task 9.1, 9.3)
- `StatusDisplay.ts` — Connection and simulation status UI (Task 10.1)
